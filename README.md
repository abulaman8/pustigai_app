# POC API for PustigAI

## Instructions

### Create a virtual env (if not already done)

```bash
python3 -m venv .venv
```

### Activate virtual env

#### Linux/Mac

```bash
source .venv/bin/activate
```

#### Windows

```bash
venv\Scripts\activate
```

### Install dependencies (if not already done)

```bash
pip install -r requirements.txt
```

### Run the API server

```bash
python main.py
```
